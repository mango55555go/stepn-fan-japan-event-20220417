## 概要
2022年4月17日に行われた `STEPN Fan Japanイベント` 参加者が参加賞としてNFTを取得できるサイト
## 技術スタック
- Next.js
- Tailwind CSS
- Supabase
  - Firebaseに代わるBaaS
- Vercel
  - Hosting
- TwitterAPI
  - TwitterIDのみを使用し、その他の情報は取得はできるが利用はしていない


## 環境構築

- `.env.local` ファイルを作成（中身はmangoからDMします）

```bash
npm install

npm run dev
```

- http://localhost:3000 でアプリで起動を確認
